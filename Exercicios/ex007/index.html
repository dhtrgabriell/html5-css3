Formatação de texto:

NEGRITO / DESTAQUE

Palavra em negrito semântico: Negrito

Palavra em negrito não semântico: Negrito

ITÁLICO / ÊNFASE

Palavra em itálico semântico: Itálico

Palavra em itálico não semântico: Itálico

Textos marcados

Podemos criar um texto marcado. Ex: Irinel! Você não sabe. Nem eu.

Texto grande e pequeno

texto grande usando big: GRANDE

texto pequeno usando small: pequeno

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mas não considerado. Ex: A terra é plana!

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Ex: Base Lunar.

Texto sobrescrito

Ex: x2+3 -> x2+3

Texto subscrito

Ex: H2O -> H2O
